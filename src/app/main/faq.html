• 淘轉是甚麼？

    淘轉是全港提供最全面的網絡平台服務公司，我們會為各會員提供一系列的服務，包括國內網購合併集運到香港，支付寶充值，香港至國內快遞，及香港至國內匯款，務求令各會員能節省最多的金錢和時間，盡情享受網上購物的樂趣。

  • 為何選擇使用淘轉合併集運？

    淘轉合併集運是為各會員可省回在國內網購運費的中轉站，如每間店鋪商家均需直接寄往香港，快遞費用往往超出物件的金額，但若寄往我們的東莞中轉站，然後再集合其他店鋪貨物一同寄往香港，節省費用約可節省高達50%。

  • 貨件可合併多少件貨品？

    淘轉物流中轉站，可作無限件合併，但基於每件不能超過每間物流公司的標準尺寸及重量，所以選擇快遞公司前，可先參考各間物流公司的標準規格。

  • 如何查詢貨件是否到達東莞中轉站？

    當您申請成為淘轉會員後，各會員便各自享有集運專屬地址，當我們收到物件後，便在系統中更新，會員各自可登入系統貨物清單內查看物流狀態。

  • 需要在網頁內認領貨件？

    只需註冊成為會員，便能登入系統。所有中轉站收到貨物後會最快時間在貨物清單內顯示，正常情況下，會員無需自行認領貨物。但如因天雨關係或特殊情況下，導致未能清晰看到會員的專屬集運地址，會員也有可能需要輸入運單編號進行認領貨件。

  • 如何得知何時取貨？

    當所有物件到達淘轉東莞中轉站簽收後，所有物流狀態均會顯示為已到物流中轉站，會員可隨時地選擇已收到的貨件進行合併。一經確認及進行付款手續，約一至三個工作天內，和淘轉合作的快遞公司便會送到會員提供的收貨地址。

  • 淘轉與快遞公司合作關係？

    淘轉物流東莞中轉站是代各會員收集貨物，再委托快遞公司作派送，所以希望各朋友仔知悉快遞公司與我們的責任，如快遞公司有任何問題，請勿在淘寶上給予我們差評，建議可直接致電給快遞公司投訴，如快遞公司也未能作出解釋，也可電郵給我們跟進。懇請各會員理解。

  • 如何計算重量？

    貨品到達國內中轉站後，我司便會計算重量，計算單位為kg，準確為0.1kg，重量總和不足一公斤按一公斤計算。

  • 怎樣計算體積重？

    貨品其中一邊的長度等於或長過35cm，我們才會計算體積重的。計算公式：長x闊x高 (cm) / 6000 ，計算結果就是重量(kg)。

  • 東莞中轉站運費可否先幫忙付款？

    東莞中轉站一律不接收運費到付，我們會退回物件到原有的快遞公司，待會員跟淘寶商家確認後，才可再把貨件送到淘轉物流中轉站。

  • 我需要另付包裝費嗎？

    本公司提供基本免費包裝服務，如有特別需要，可透過阿里旺旺與我們職員聯絡。

  • 一般貨件可在中轉站存倉多久？

    貨到中轉站，由簽收日起計，我司會提供長達六十天免費存倉時間（*大型貨件除外）。如超過免費存倉時間，每天會收取每個快遞單號人民幣五元存倉費用。例如: 合併單號內裡的其中一個單號超出一天，需要存倉費為人民幣五元，如有兩單，存倉費則為人民幣十元，如似類推 …

  • 速遞貨品有限制嗎？

    匯通物流貨品長度不可超過100cm，單件貨品不可以重過30公斤。
    倚天物流貨品長度不可超過250cm，單件貨品不可以重過30公斤。
    運通物流貨品長度不可超過130cm，單件貨品不可以重過26公斤。
    ( 如超過以上尺寸，可先通知我們查詢，我們會作另外派送 )
    另據承運契約條款，嚴禁承運違禁品，包括: 液體、粉末、冒牌貨品及任何侵權物品等，違者一律送官究治，有關戶口亦會被即時禁止。

  • 貨品在內地簽收後，淘轉何時會把貨品資料更新？

    貨品在內地成功簽收後，中轉站會於辦公時間內更新資料。遇上旺季或特別節日，更新時間會有所延長，敬請見諒。

  • 需時多久才可送貨？

    一般情況是1-3天便可送貨到指定地址。除了海關查車例外，所以時間並不能百分百保證，敬請見諒。

  • 可否在簽收貨品時替我先檢查貨品？

    為免有不必要的誤會，所有貨品都不會檢查的，只會保留原本商戶發貨的包裝。而外包裝有明顯破損的物品，我們會代會員拒絕收貨，避免責任上的問題。
